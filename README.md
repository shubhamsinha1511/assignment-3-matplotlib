📊 Data Visualization Assignment (Matplotlib + Pandas)

This project is part of Assignment 3, focused on practicing data visualization using Python libraries like Matplotlib, Pandas, and NumPy.

It demonstrates multiple types of plots commonly used in data analysis and reporting.

🚀 Tech Stack
Python 🐍
Matplotlib
Pandas
NumPy
📌 Features / Visualizations Covered

This assignment covers a wide range of plots:

1. 📈 Line Chart (Stock Prices)
Visualizes stock price trends of Alphabet Inc.
Compares Open, High, Low, and Close values
2. 📊 Horizontal Bar Chart
Displays relationship between number of students and marks
3. 📅 Year-wise Sales Analysis
Compares sales across multiple years (2018–2021)
4. 🎨 Customized Line Chart
Demonstrates styling (markers, dashed lines, thickness)
5. 🏢 Bar Chart (Employee Data)
Department-wise employee distribution
6. 🌍 Scatter Plot + Bar Graph
Scatter: Population vs Literacy rate by region
Bar: Average literacy rate per region
7. 📦 Box Plot
Subject-wise performance distribution
8. 👥 Grouped Bar Chart
Comparison of scores between men and women
9. 🥧 Pie Chart
Product-wise sales percentage distribution
9(b). 🔲 Subplots
Multiple charts in a single figure (Soap vs Facewash sales)
10. 📉 Histogram
Profit distribution analysis
